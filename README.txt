Телеграм Чат-бот "Медицинская инициатива".

- ссылка на бота: t.me/med_inicia_bot

1. Инструкция для запуска:(Использованное ПО: VS Code, IDE WebStorm. Используемые библиотеки: node-telegram-bot-api, nodemon, @emailjs/nodejs)
	Ввести команды в консоли в корневой директории бота, в которой расположен файл index.js:
	1) npm init
	2) npm install - установка библиотек
	3) node index.js - запуск бота


2. Краткое описание структуры:

index.js - входной файл, где описываются основные направления работы бота (обработка кнопок главного меню, инициализация emailjs)
callbacks.js - обработчики нажатий на все кнопки в inline-keyboard (все описания услуг и отделений находятся)
controllers.js - функции для обработки нажатия на "Оценка", "Задать вопрос", "Записаться на приём"
menus.js - большая часть меню(inline-keyboard, reply_markup)


3. Приём данных от пользователей:

Запись в файл (производится в controllers.js, на данный момент закоментирована и не выполняется):
	reviews.txt - оценки и отзывы
	register.txt - запись на талоны
	question.txt - вопросы

Отправка на почту: doctor@131.by (передается в аргументе 'to' объекта templateParams)

Сервис для отправки почты: Emailjs(https://www.emailjs.com/)

По умолчанию письма будут отправляться с:

Письма отправляются в файле controllers.js при помощи метода emailjs.send():
Инициализация проводится в файле index.js
Инструкция к методу отправки: https://www.emailjs.com/docs/sdk/send/


4. Изображения
Изображения хранятся по пути .assets/services/
